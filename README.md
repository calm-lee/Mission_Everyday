## 미션! 에브리데이 소개

<b>1. 프로젝트 개발 목적</b><br>
작심 3일로 끝나기 쉬운 자기계발에 동기를 부여하고자 개발한 1인 프로젝트입니다.<br>

<b>2. 개발 기간</b><br>
2021.09.02 - 2021.10.02 (1개월)<br>

<b>3. 수행 역할 (1인 프로젝트)</b><br>
* 기획서, 프로젝트 일정표 작성
* DB, URL, API 설계
* 개발환경 구축
* 프론트엔드
* 백엔드
* 트러블슈팅
* 동작 테스트
* AWS 배포

<b>4. 미션!에브리데이 주요 기능</b>
* 운동, 공부, 외국어 등  다양한 카테고리의 미션 가입
* 인증 포스트 올리기  (사진  첨부  필수)
* 좋아요, 댓글달기 가능
* 마이페이지에서 내 인증현황 확인 가능

<b>5. 사용 기술 스택</b>
* 백엔드: Java, Spring Boot 4.11, Apache Tomcat 9.0 
* 프론트엔드: JavaScript, Ajax, HTML, CSS, jQuery, BootStrap
* 데이터베이스: MySQL, Mybatis
* 배포: AWS
* 형상관리: Git, Sourcetree

## 프로젝트 설계 과정
### 1. 기획서
> 카카오 오븐을 통해 [기획서](https://ovenapp.io/project/u1vPXyC6FPiBUZFJYClPuW5lQGtSjUf9#1HJPw)를 설계했습니다.  

### 2. DB 설계
> 사용 용도에 맞게 다음과 같이 DB를 설계했습니다.

 1) user : 로그인, 회원가입, 마이페이지에  사용
 2) category : 카테고리, 미션 페이지에 사용
 3) mission : 상세 미션 페이지에 사용
 4) mission_user : 미션 별 유저의 수를 조회하는 데 사용 (관계 테이블)
 5) post : 포스트 생성/수정/삭제에 사용
 6) comment : 댓글 생성/삭제에 사용
 7) like : 좋아요 생성/해제에 사용

### 4. 환경 구축
>Spring Boot로 프로그램을 설계하고 Apache Tomcat을 통해 웹 서버로 연결했습니다.
<img src="https://img1.daumcdn.net/thumb/R1280x0/?scode=mtistory2&fname=https%3A%2F%2Fblog.kakaocdn.net%2Fdn%2FcxA4wo%2FbtrlpVKUCiy%2FVupLCCxdXvdOKZukq0GHDK%2Fimg.png" alt="setting" width="480">  


>MVC pattern을 통해 Model<->View(JSP)<->Controller가 상호작용하도록 처리했습니다.
<img src="https://img1.daumcdn.net/thumb/R1280x0/?scode=mtistory2&fname=https%3A%2F%2Fblog.kakaocdn.net%2Fdn%2FMjNpH%2FbtrloyCFNcm%2Fh5KqEvlMxqErWJN7uebOZk%2Fimg.png" alt="mvc" width="480">

* JSP: View 화면 구축
* JavaScript: 동적 화면 구축
* AJAX: Data, Request 전달
* Java: Controller, Service, Repository로 Request/Response 처리
* MyBatis: Mapper로 MySQL에 쿼리 실행
